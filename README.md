# BenchTest

Bench REST Test app

## Viewing

For a quick preview without building anything, simply open `dist/index.html`.

## Installation

- Install [Node.js](https://nodejs.org/en/download/)
- Install [Yarn](https://yarnpkg.com/en/docs/install)
- Run `yarn` in this folder.

## Run Dev Server

```sh
yarn run start
```
